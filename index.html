Base

A very minimalistic collection of CSS I like to use as defaults

Font

.base {
font-family: Helvetica, Arial, Sans-Serif;
font-size: 15pt;
}

Text Input

<input type="text"></input>
<input class="base" type="text"></input>

Buttons

<input type="button" class="base" ... is interchangeable with <button class="base" ...

<input type="button" value="Button"></input>
<input class="base" type="button" value="Default"></input>
<input class="base green" type="button" value=".green"></input>
<input class="base red" type="button" value=".red"></input>
<input class="base" type="button" value="disabled" disabled></input>

Button Sizing

<input class="base" type="button" value="Default"></input>
<input class="base medium" type="button" value=".medium"></input>
<input class="base large" type="button" value=".large"></input>
